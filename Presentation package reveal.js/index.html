HTML / CSS

  • Un framework : Laravel
  • Type MVC (Modèle Vue Contrôlleur) -> Modèle définit les objets (ex les etudiants), Vue définie ce qu'on voit (contient l'HTML notamment et complète avec les données fournies par le contrôlleur), et le fameux contrôlleur qui définie toute la logique (on lui donne des informations, il les traites et les redonnes le plus souvent à la vue)
  • Passé le temps de d'apprentissage -> gain de temps
  • Laravel Scout (par Laravel) -> Envoie les données formatées à Algolia
  • 1x DEVELOPPER
  • Eloquent (ORM = Doctrine) -> Pour gérer la base de données très facilement
  • Ex : Obtenir les infos de l'entreprise liée à l'offre. On définie qu'une offre appartient donc à une entreprise (en accord avec la BDD). On a plus qu'ensuite à utiliser les méthodes "magiques" (leur vrai nom) de Laravel pour obtenir l'information que l'on souhaite.
